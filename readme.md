# Flighty flight controller

this is my first flight controller that i designed (so happy :) it will be used to control quadcopter , uav , rockets etc... personally i will use it for my rc vtol uav 

## FEATURES 

* 7x servo spot ( for rc plane purpose or drone)
which 2 can be used as fire lighter ( rocketry )
* Can be powered up by 2s lipo / li ion battery 
* Use the best m4 cortex mcu available which is the f405rgt6
* Easy to solder a lot of THT component and is very educational ( its big )
* Simple to repair 
* Sd-card to re-analyse your past crash ( to see what happend )
* And finally its cool



## software 
You can use Ardupilot or Inav once flashed in the flightcontroller but you need to make a target in the designated firmware ( betaflight or ardupilot or inav )
